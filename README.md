# TFC Helpers

TFC Helpers - веб-приложение с калькуляторами для TerraFirmaCraft.
```text
https://kopnu.github.io/tfc-helpers/
```

### Возможности

- Калькулятор сплавов с проверкой допустимых пропорций.
- Подбор валидного рецепта с учетом доступных порций металла.
- Калькулятор крицы и стали для расчета железосодержащих предметов, флюса и угля.
- Помощник Prospector's Pick для поиска вероятной области руды по истории замеров.
- Переключение языка интерфейса: русский и английский.

### P.s.

Разрабатывалось с помощью ИИ агента

## English

TFC Helpers is a web app with calculators for TerraFirmaCraft.

### Features

- Alloy calculator with valid proportion checks.
- Valid recipe search with available metal portions.
- Bloom and steel calculator for iron-bearing items, flux, and charcoal.
- Prospector's Pick helper that narrows down the probable ore area from sample history.
- Interface language switcher: Russian and English.

### P.s.

It was developed with the help of an AI agent
